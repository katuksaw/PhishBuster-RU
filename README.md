# 🛡️ PhishBuster-RU

Браузерное расширение для выявления фишинговых сайтов на основе анализа цифровых метрик и локальной базы эталонов.

## 📌 Описание проекта

PhishBuster-RU анализирует веб-страницы в реальном времени, сравнивая их цифровые характеристики с эталонными данными легитимных российских сервисов (Госуслуги, Сбербанк, ВКонтакте и др.).

### 🔍 Что проверяет расширение:
- Структуру DOM-дерева страницы
- Наличие официальных CSS-классов и элементов
- Данные SSL-сертификата
- Доменные паттерны
- Фавиконы и мета-теги

## 🚀 Возможности

✅ Локальная обработка данных (без отправки на сервер)  
✅ Мгновенная проверка при загрузке страницы  
✅ Цветовая индикация риска (зелёный/жёлтый/красный)  
✅ Объяснимые предупреждения с указанием причин  
✅ Поддержка Manifest V3
