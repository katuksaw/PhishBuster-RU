# PhishBuster RU 🛡️

**Браузерное расширение для выявления фишинговых сайтов на основе анализа цифровых метрик и локальной базы эталонов**

## О проекте

PhishBuster RU — это расширение для Google Chrome, которое анализирует веб-страницы в реальном времени и определяет, является ли открытый сайт фишинговым. Расширение использует три независимые метрики:

1. **Доменные паттерны** — проверка домена по белому списку и подозрительным паттернам
2. **DOM-селекторы** — поиск фирменных элементов интерфейса
3. **Ключевые слова** — анализ заголовка и текста страницы

Результат вычисляется с помощью **весовой аддитивной модели**:

```
Score = 40 × m_domain + 35 × m_selector + 25 × m_keyword + 15 × m_urgency
```

## Установка

1. Скачайте или клонируйте репозиторий
2. Откройте Chrome → `chrome://extensions/`
3. Включите «Режим разработчика» (переключатель справа вверху)
4. Нажмите «Загрузить распакованное расширение»
5. Выберите папку `phishbuster/`

## Как использовать

1. **Автоматический анализ**: расширение анализирует каждую открываемую страницу
2. **Предупреждения**: при обнаружении фишинга на странице появляется красный индикатор
3. **Детали**: нажмите на иконку расширения для просмотра подробностей анализа

## Алгоритм скоринга

| Метрика | Вес | Описание |
|---|---|---|
| Доменный паттерн | 40 | Проверка домена по whitelist и suspicious patterns |
| DOM-селекторы | 35 | Поиск фирменных элементов интерфейса |
| Ключевые слова | 25 | Анализ title и body на брендовые слова |
| Urgency-сигналы | 15 | Бонус за обнаружение социальной инженерии |

### Пороги вердиктов

| Score | Вердикт | Цвет |
|---|---|---|
| 0–29 | safe | 🟢 Зелёный |
| 30–59 | warning | 🟡 Жёлтый |
| 60–100 | danger | 🔴 Красный |
